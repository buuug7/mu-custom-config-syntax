# MU Server Custom Config Syntax

This extension provides syntax highlighting for MU server custom config files under `Data/Custom`.

## Features

- Highlights `//` comments
- Marks section header lines containing a single number
- Recognizes the `end` keyword
- Highlights numeric values and item identifiers like `02,001`

## Usage

1. Open a file under `Data/Custom`, such as `CustomLottery.txt`.
2. If the language does not auto-detect, open the language picker and select `MU Custom Config`.
3. Install the extension by opening the `mu-custom-config-syntax` folder as a workspace or using VS Code's extension development host.
